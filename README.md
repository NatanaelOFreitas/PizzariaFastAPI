# 🍕 Pizza Delivery API

API REST para gerenciamento de pedidos de pizzaria, focada em segurança e performance.

## 🛠️ Tecnologias
- **FastAPI** (Python)
- **SQLAlchemy** (Banco de Dados/ORM)
- **JWT & Passlib** (Autenticação e Criptografia)
- **Pydantic** (Validação de dados)

## 🔑 Principais Recursos
- **Autenticação:** Sistema de login e proteção de rotas com JWT.
- **Pedidos:** Fluxo completo de criação e gerenciamento de pedidos.
- **Segurança:** Criptografia de senhas (hashing).
- **Docs:** Documentação automática via Swagger UI (`/docs`).
